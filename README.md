# Technical challenge - Frontend

This project was created with the objective of the user to manage their categories and products. See all features below.

## Features

#### Categories
- List of all categories
- Get a category by id
- Add a new category
- Edit an existing category by id
- Delete an existing category by id
#### Products
- List of all products
- Get a product by id
- Add a new product
- Edit an existing product by id
- Delete an existing product by id

## Stack
- React
- Typescript
- Styled-components
- ContextAPI
- Axios
- Zod
- Phosphor Icons


## Running locally

Install dependencies

```bash
  npm install
```

Starting the server

```bash
  npm run dev
```
